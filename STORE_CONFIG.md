## Deployment Workflows

### Oxygen (Primary)
- **Trigger:** Push to any branch
- **Production:** Pushes to `main` branch
- **Preview:** Pushes to any other branch
- **URL:** Automatically generated by Oxygen
- **Management:** Shopify Admin → Sales Channels → Hydrogen

### Vercel (Secondary)
- **Trigger:** Push to any branch
- **Production:** Pushes to `main` branch  
- **Preview:** Pull requests and branch pushes
- **URL:** `artfulfam.vercel.app` (or custom domain)
- **Management:** vercel.com dashboard

## Local Development

Start the development server:
```bash
npm run dev
```

The local server will:
- Use MiniOxygen (local Oxygen emulator)
- Connect to your production Shopify store
- Run on `http://localhost:3000`
